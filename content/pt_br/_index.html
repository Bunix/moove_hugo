{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} O Moove é um tema do Moodle que torna os anbientes virtuais de aprendizagem mais limpos e intuitivos para os estudantes e educadores. Seu layout intuitivo é otimizado para ensino online, focando nas coisas que importam, suas atividades de ensino e conteúdos. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-brain" title="Foco no que importa!" %}} O tema permite que os usuários ocultem as colunas laterais, maximizando a área de conteúdo, foncando no que interessa, seu conteúdo principal. {{% /blocks/feature %}} {{% blocks/feature icon="fa fa-users" title="Guiado pela comunidade" %}} Sempre aberto a novas idéias e sugestões. Tentando manter um diálogo aberto com a comunidade para resolver bugs, desenvolver novas funcionalidades, além de melhorar as existentes. {{% /blocks/feature %}} {{% blocks/feature icon="fa fa-universal-access" title="Acessibilidade" %}} O tema é o único que contém funcionalidades de acessibilidade. Acredito que a educação é para todos, portanto não podemos deixar niguém de fora. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section color="white" >}}
