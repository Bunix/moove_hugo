--- title: "Installation" linkTitle: "Installation" weight: 3 description: > How to install the theme into your Moodle ---

First way

  • Find the right theme version for you moodle version from moodle.org site
  • Download the plugin code
  • Extract the content
  • Put the folder into the folder theme of your moodle
  • Access the notification area in moodle and install

Second way (from github)

  • Clone the theme's repository into the folder theme.
  • Access the notification area in moodle and install
